# Notes for C

## Notes

- When printing strings, we must add a newline (`\n`) character so that our next `printf` statement will print in a new line.

## Shortcut

- comment: `Ctrl + /`

## statements

- `int`: return a int
- `void`: don't return a value

## define boolean type

```c
#define BOOL char
#define FALSE 0
#define TRUE 1
```

## define variables

```c
int foo;
int bar = 1;
```
